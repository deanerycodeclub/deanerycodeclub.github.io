Deanery Code Club

Code Club is a weekly event at the Deanery, attended by a number of year 5 and 6 pupils who are interested in learning to code.

You can find out more about Code Club, which is a worldwide organisation setup to help people learn to code, by going here: Code Club

What languages do we use?

We've been generally following the design of the Code Club course projects and that has meant we've been using:

  • Scratch
  • HTML and CSS

We plan to use other languages soon.
